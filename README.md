Hostel Management System 

Programu hii imeandikwa kwa kutumia **Python** na **SQLite** ili kusaidia kusimamia wanafunzi wanaoishi kwenye hostel.  
Mfumo huu ni rahisi na unaweza kupanuliwa kulingana na mahitaji yako.

## Vipengele vya Programu
 **[Ongeza mwanafunzi](ca://s?q=Ongeza_mwanafunzi_katika_hostel_program)**: Kuhifadhi jina na namba ya chumba cha mwanafunzi.
 
 **[Onyesha wanafunzi](ca://s?q=Onyesha_orodha_ya_wanafunzi_hostel)**: Kuorodhesha wanafunzi wote waliopo kwenye database.
 
 **[Futa mwanafunzi](ca://s?q=Futa_mwanafunzi_katika_hostel_program)**: Kuondoa mwanafunzi kwa kutumia ID yake.

## Teknolojia Zilizotumika
Python 3
SQLite3 (database nyepesi inayohifadhi data kwenye faili `hostel.db`)


##  Jinsi ya Kutumia
1. Hakikisha una Python 3 imewekwa kwenye kompyuta yako.
2. Pakua au nakili faili ya `hostel.py`.
3. Endesha programu kwa amri:
   ```bash
   python hostel.py
